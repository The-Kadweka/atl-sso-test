Make customers free from remembering testimonials in Odoo!!

Single sign-on allows website portal users to login with credentials of one primary website to multiple secondary websites. It never compromises with the safety of the user data as it uses OAuth.
OAuth is used to grant websites access to their information on other websites but without giving them the passwords.
Odoo Microsoft SSO facilitates you to enable the single sign-on feature in your Microsoft website. Hence, your customers can log into their Odoo website accounts with the credentials of the Microsoft Website Account.

Detailed Features List

Make sign-on easy for your customers

  • Your customers can sign-on to all your Odoo websites just by a single click.
  • Customers can log in to all your configured Odoo websites if they have Microsoft accounts with the same credentials.

No Compromise With The Security

  • Login authentication is secure with the help of Odoo Microsoft SSO.
  • All the customer logins will be tracked for security to avoid cyber frauds.

Save your time as well as money

  • It helps you to save IT help desk cost
  • No need to enter the credentials again and again; just a click can help to login

Manage Customers from the Odoo backend

  • The customers will be automatically added on Odoo backend after signing in.
  • As an admin, you can add or delete the customer from Odoo Backend.
